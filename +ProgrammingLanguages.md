# Language Comparisons

## Arraylists/Vectors/Dynamic Arrays

### 0. Libraries / Instantiation
- C++
    ```c++
    #include <vector>;

    vector<int> = a;
    ```
- Python
    ```python
    a = [] #Dynamic Arrays are built-in
    ```
- JavaScript
    ```js
    let a = []; //Also built-in
    ```

### 1. Adding Elements to the End of an ArrayList:
- C++
    ```c++
    a.push_back(23);
    ```
- Python
    ```python
    a.append(32)
    ```
- JavaScript
    ```js
    a.push(32);
    ```
