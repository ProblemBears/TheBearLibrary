## 1. Dynamic Arrays

<div id='1-a'/>

### Instantiation/Libraries
- C++
    ```c++
    #include <vector>;

    vector<int> = a;
    ```
- Python
    ```python
    a = []              #built-in
    ```
- JavaScript
    ```js
    let a = [];         //built-in
    ```

<div id='1-b'/>

### Push
- C++
    ```c++
    a.push_back(23);
    ```
- Python
    ```python
    a.append(32)
    ```
- JavaScript
    ```js
    a.push(32);
    ```
